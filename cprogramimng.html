C Programming Questions Sheet — I Semester

Prepared By: Prof. Nadeem Khan, BIRT/S Bhopal

Below is a clean, printable list of 52 beginner-to-intermediate C programming exercises. Use this for assignments, practice sheets, or classroom handouts.

  1. WAP to read length and width of a rectangle. Print the area.
  2. WAP to read base and height of a triangle. Find the area.
  3. Find the last digit of a given number.
  4. WAP to read a number and find the last 2 digits of the given number.
  5. WAP to read a number and find the last 3 digits of the given number.
  6. WAP to read a number and remove the last digit of the given number.
  7. WAP to read a number and remove the last 2 digits of the given number.
  8. WAP to read a number and remove the last 3 digits of the given number.
  9. WAP to find the bigger number between 2 numbers.
  10. WAP to find the biggest number among 4 numbers.
  11. WAP to read the radius of a circle and print the area.
  12. WAP to find Simple Interest.
  13. WAP to read basic, HRA, DA, and PF. Find the Net Salary.
  14. WAP to read 2 numbers and swap the numbers.
  15. WAP to swap 2 numbers without using a third variable.
  16. WAP to swap 2 numbers using one line of code.
  17. WAP to read a lowercase character and convert it to uppercase.
  18. WAP to read an uppercase character and convert it to lowercase.
  19. WAP to find the sum of N natural numbers without using any loop.
  20. WAP to find the bigger number between 2 numbers without using if-else or any conditional operator.
  21. WAP to find the smaller number between 2 numbers without using if-else or any conditional operator.
  22. WAP to read a number and print whether it is odd or even without using the '%' symbol.
  23. WAP to read a number and print whether it is positive or negative without comparing with 0.
  24. WAP to read purchase rate and sales rate. Print whether it is profit or loss.
  25. WAP to read a character and check whether it is lowercase or not.
  26. WAP to read a character and check whether it is uppercase or not.
  27. WAP to read a character and check whether it is a digit or not.

  4. WAP to read a character and print whether it is a vowel or consonant.

  21. WAP to read a week number and print the corresponding weekday.
  22. WAP to read 2 numbers and an operator; find the result based on the operator.
